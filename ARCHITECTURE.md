# Архитектура Vanilla

Основной код находится в `vanilla_bot.py`. Он:

- разбирает параметры запуска;
- читает параметры инструментов из `SBER.json` и `CNYRUB_TOM.json`;
- считает сторону, цену и количество лотов;
- подключается к RabbitMQ;
- создаёт четыре временные очереди для статусов, сделок, тиков и стаканов;
- ждёт нужные рыночные данные для `slippage` или `best_quote`;
- отправляет одну лимитную заявку;
- ждёт `orderId` и финальный статус `Filled` или `Cancelled`;
- собирает сделки по `orderId`;
- печатает итоговую строку.

Очереди имеют уникальные имена с `owner`, ограничение `x-max-length = 1000` и политику `drop-head`. В конце работы consumers отменяются, очереди удаляются, соединение закрывается.

```text
CLI -> RabbitMQ -> заявка -> статусы и сделки -> итоговая строка
```
